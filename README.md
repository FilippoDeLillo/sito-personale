# Portfolio Web | Filippo De Lillo

Il mio sito web personale, sviluppato per essere una vetrina minimalista, ultra-performante e focalizzata sull'esperienza utente. 

Questo progetto nasce con l'obiettivo di offrire una presentazione chiara e diretta delle mie competenze , ponendo l'efficienza del codice al centro di tutto.

## 🚀 Obiettivi del progetto
- **Performance estrema:** Minimizzare i tempi di caricamento evitando framework pesanti.
- **Accessibilità:** Garantire una fruizione fluida su ogni dispositivo.
- **SEO-First:** Struttura semantica ottimizzata per i motori di ricerca.
- **Privacy-by-design:** Gestione dei dati conforme al GDPR senza l'uso di banner invasivi.

## 🛠 Tecnologie Utilizzate
- **HTML5:** Struttura semantica rigorosa.
- **CSS3:** Layout moderno, mobile-first con variabili CSS e animazioni native.
- **JavaScript (Vanilla):** Gestione della logica, della Dark Mode e delle interazioni allo scorrimento (Intersection Observer API).
- **Formspree:** Backend per la gestione sicura del modulo di contatto.

## 📊 Performance
Il progetto è stato testato con **Google Lighthouse**, ottenendo i seguenti punteggi (Audit Mobile):

- **Performance:** 99/100
- **Accessibility:** 90/100
- **Best Practices:** 100/100
- **SEO:** 100/100

## 📝 Metodo di lavoro
Credo che la tecnologia debba risolvere problemi, non crearne di nuovi. Il mio processo si divide in tre fasi:
1. **Analisi:** Identificazione delle necessità reali del business.
2. **Sviluppo:** Scrittura di codice pulito, modulare e orientato all'efficienza.
3. **Consegna:** Formazione del cliente per garantire piena autonomia operativa.
